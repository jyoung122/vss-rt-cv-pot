## Module design — `backend/app/nvstreamer.py`

Replaces `sdr.py` (37 lines). Functions:

- `async def register_sensor(name: str, rtsp_url: str) -> str`  
  `POST /api/v1/sensor/add` with `{"name":name,"sensorUrl":rtsp_url,"location":"","tags":"","username":"","password":""}`. Returns `sensor_uuid`. Note: `tags` must be string not array — array crashes NVStreamer with `Json::LogicError` (validated on 2026-05-12 EC2 run).

- `async def unregister_sensor(sensor_uuid: str) -> None`  
  `DELETE /api/v1/sensor/<sensor_uuid>`. Logs warning on non-2xx; does not raise.

- `async def get_proxy_url(sensor_uuid: str) -> str | None`  
  `GET /api/v1/sensor/list`, finds entry by `sensorId == sensor_uuid`, returns `rtsp://<nvstreamer-host>:30554/live/<sensorId>`. Returns `None` if not found or list unreachable.

- `async def list_sensors() -> list[dict]`  
  `GET /api/v1/sensor/list`. Returns raw list; for ops/debugging.

Env vars: `NVSTREAMER_URL` (default `http://nvstreamer:30000`). Uses `httpx.AsyncClient(timeout=10)` pattern from existing `sdr.py:12`.

## Module design — `backend/app/rtsp_publisher.py`

New module. Functions:

- `def start(video_id: str, file_path: Path) -> str`  
  Spawns `ffmpeg -re -stream_loop -1 -i <file_path> -c copy -f rtsp rtsp://mediamtx:8554/<video_id>`. Stores `Popen` in `_publishers[video_id]` under `_lock`. Returns the RTSP URL string.

- `def stop(video_id: str) -> None`  
  Sends `SIGTERM` to process; calls `process.wait(timeout=5)`. Removes from `_publishers`. Logs if already gone.

- `def is_alive(video_id: str) -> bool`  
  Returns `True` if `video_id` in `_publishers` and `process.poll() is None`.

Internal state: `_publishers: dict[str, subprocess.Popen] = {}` with a `threading.Lock`. Env var: `MEDIAMTX_URL` (default `rtsp://mediamtx:8554`).

ffmpeg flags rationale: `-re` (real-time rate for RTSP), `-stream_loop -1` (indefinite loop until `stop()`), `-c copy` (no re-encode — trivial CPU cost per the rediagnosis log).
